Introduction

Welcome to the official documentation for ExophsBot, a Minecraft bot designed for the 5B5T server. This bot provides various utility, fun, movement, and communication commands to enhance your gameplay experience. All public commands are prefixed with ! and can be used in in-game chat.

Note: This page lists public commands only. Admin commands (starting with :) are not shown here. For questions or suggestions, join our Discord using !discord.

Command Usage Description
!help !help Displays a link to this documentation page.
!ping !ping [username] Shows your ping or the ping of another player. Returns "unknown" if not available.
!lastmsg !lastmsg <username> Shows the most recent message sent by the specified player.
!firstmsg !firstmsg <username> Shows the first recorded message sent by the specified player.
!randommsg !randommsg <username> Picks and shows a random message from the specified player's message history.
!info !info Displays information about the bot's runtime environment (Node.js version, OS, CPU).
!randomn !randomn <decimals> <min> <max> Generates a random decimal number between min and max with the specified precision (0–99 decimals).
!prefix !prefix Shows the current command prefix (currently "!").
!uptime !uptime Shows how long the bot has been running (in minutes).
!online !online Shows the current number of players online on the server.
!discord !discord Displays the invite link to the official Discord server.
!gay !gay [target] Shows a random funny percentage of how "gay" you or another player are (joke command).
!coords !coords Shows completely fake random coordinates (for memes/trolling).
!pitch !pitch Shows the bot's current camera pitch value.
!yaw !yaw Shows the bot's current camera yaw value.
!doxx !doxx [target] Shows a fake random IP address (purely for jokes — not real doxxing).
!health !health Shows the bot's current health and hunger values.
!hunger !hunger Shows the bot's current hunger and health values.
!y/n !y/n Answers your question with a random YES or NO.
!roast !roast <username> Sends a random savage/funny roast directed at the specified player.
!kit !kit <name> Fake kit claim message (currently just cosmetic).
!rules !rules Shows basic server rules and how to report players.
!report !report <player> [reason] Creates a fake/public report message for the specified player.
!8ball !8ball <question> Ask the Magic 8-Ball a question and get a random classic response.
!offlinemsg !offlinemsg <player> <message> Sends an offline message to the specified player. They will see it when they join (or sometimes when they chat).
!ignore !ignore <player> Blocks a player from sending you offline messages via !offlinemsg.
!unignore !unignore <player> Removes a player from your ignore list so they can send you offline messages again.
!move !move <forward/back/left/right/jump/stop/look> [time or look params] Makes the bot walk, jump, stop, or look in a direction. Supports timed movement and look yaw/pitch.
!goto !goto <x> <y> <z> Makes the bot pathfind to the specified coordinates.
!come !come Makes the bot walk toward the player who used the command.
!follow !follow <player> Makes the bot continuously follow the specified player.
!attack !attack <player/animal/stop> Makes the bot attack a player or certain animals with crits. Use "stop" to cancel.
