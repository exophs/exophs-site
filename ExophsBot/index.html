Introduction Commands List

Introduction

Welcome to the official documentation for ExophsBot, a Minecraft bot designed for the 5B5T server. This bot provides various utility, fun, and movement commands to enhance your gameplay experience. All commands are prefixed with ! and can be used in the in-game chat.

Note: This documentation excludes admin-only commands. For any issues or suggestions, join our Discord server using the !discord command in-game.

Commands List

Command Usage Description
!help !help Displays a list of all available commands with brief descriptions.
!ping !ping [username] Gets the ping (latency) of the specified user or yourself if no username is provided. Returns "unknown" if not available.
!lastmsg !lastmsg <username> Retrieves the last message sent by the specified user.
!firstmsg !firstmsg <username> Retrieves the first message sent by the specified user.
!randommsg !randommsg <username> Retrieves a random message from the specified user's message history.
!info !info Displays system information including Node.js version, operating system, and CPU model.
!randomn !randomn <decimals> <min> <max> Generates a random number between min and max with the specified number of decimal places (0-99).
!prefix !prefix Displays the current command prefix ("!").
!uptime !uptime Shows the bot's uptime in minutes.
!online !online Displays the number of players currently online.
!discord !discord Provides the link to join the Discord server: https://discord.gg/dN8jCQymN7
!gay !gay [target] Generates a humorous random percentage of how "gay" the target (or yourself) is.
!coords !coords Generates and displays fake random coordinates in the Overworld.
!pitch !pitch Displays the bot's current camera pitch.
!yaw !yaw Displays the bot's current camera yaw.
!doxx !doxx [target] Generates and displays a fake IP address for the target (or yourself) as a joke.
!health !health Displays the bot's current health and hunger levels.
!hunger !hunger Displays the bot's current hunger and health levels.
!y/n !y/n Randomly responds with "YES" or "NO".
!roast !roast Delivers a random humorous roast message to the user.
!kit !kit <name> Simulates receiving a kit with the specified name.
!rules !rules Displays the server rules and how to report offenders.
!report !report <player> [reason] Reports the specified player, optionally with a reason.
!8ball !8ball <question> Provides a random Magic 8-Ball response to your question.
!move !move <forward/back/left/right/jump/stop/look> [time in seconds or look params] Controls the bot's movement in the specified direction for an optional duration, or adjusts look direction (e.g., yaw/pitch or directions like forward/up).
!goto !goto <x> <y> <z> Moves the bot to the specified coordinates using pathfinding.
!come !come Makes the bot move towards the player who issued the command.
!follow !follow <player> Makes the bot follow the specified player.
!attack !attack <animal/playername/stop> Makes the bot attack the specified target (animal or player) with critical hits, or stop attacking if "stop" is used. Limited to certain animals.
